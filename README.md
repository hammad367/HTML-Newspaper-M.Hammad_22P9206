## Project Description
This project recreates a pre-1990 Urdu newspaper front page using only HTML. The layout, headlines, and structure were carefully designed to visually resemble the original newspaper while following HTML5 standards.

## Features Implemented
- Inline styling only (no external CSS)
- Multi-column newspaper layout
- Proper heading hierarchy
- Images with alt attributes
- Sectioned content layout
- Structured header and footer design

## Challenges Faced
The main challenge was recreating a complex newspaper layout using only HTML without external CSS. Managing spacing and column alignment required careful structuring and inline styling.
